VM Translator : Traduction des fichiers VM en assembleur Hack.

Jack Compiler : Compilation des programmes Jack en code machine Hack.

OS : Développement d'un système d'exploitation minimaliste pour la machine Hack.

Exemple d'utilisation:

Traduction d’un fichier VM:

1)Créez un fichier .vm avec le code intermédiaire.
2)Exécutez le VM Translator
3)Un fichier .asm sera généré, prêt à être exécuté sur la machine Hack.

Compilation d’un programme Jack:
Créez un fichier .jack
Lancez le Jack Compiler
Le code machine généré pourra être exécuté.

Test de l’OS
Chargez le système d’exploitation dans l'émulateur Hack.
Utilisez les programmes Jack ou VM pour interagir avec l'OS.
